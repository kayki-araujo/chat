API Single Chat

Este projeto é uma simples API de troca de mensagens, fornecendo registro de mensagens e retorno de mensagens com banco de dados MySQL. Projeto apresentado como exemplo nas aulas de criação de site do CIMOL

Ler mensagens

GET

http://localhost/web/chat/

Objeto

{timestamp:159753684}

Postar mensagens

POST

http://localhost/web/chat/

Objeto

{nick:'Zico', message:'Olá'}
